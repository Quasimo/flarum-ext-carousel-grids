# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-03-27

### Fixed
- Mobile responsiveness: Now displays 1 item on mobile (≤768px) and 2 items on tablet (769-1024px)
- Carousel now correctly shows only visible columns instead of stacking all items

### Added
- Pause carousel animation on hover
- Responsive column adjustment based on screen size

## [1.0.0] - 2026-03-26

### Added
- Initial release
- Grid layout with configurable columns (1-6)
- Auto-scroll carousel with fade transitions (5s interval)
- Background images and optional logos
- Drag & drop reordering in admin panel
- Display position options (after hero / before footer)
- Display scope options (homepage only / all pages)
- Responsive design (mobile: 1 column)
- English and Chinese (Simplified) translations
