# Carousel Grids

A Flarum extension that displays carousel grids with background images, logos, and auto-scroll.

## Features

- Grid layout with configurable columns (1-6)
- Auto-scroll carousel
- Background image + optional logo per item
- Responsive design (mobile: 1 column)
- Click to open links in new tab

## Installation

```bash
composer require your-vendor/flarum-ext-carousel-grids
```

## Usage

1. Enable the extension
2. Go to Admin → Carousel Grids
3. Set columns per row
4. Add grid items with title, description, images, and links
5. Drag to reorder items
