# Carousel Grids

![License](https://img.shields.io/badge/license-MIT-blue.svg)
[![Latest Stable Version](https://img.shields.io/packagist/v/quasimo/flarum-ext-carousel-grids.svg)](https://packagist.org/packages/quasimo/flarum-ext-carousel-grids)

A Flarum extension that displays carousel grids with background images, logos, and auto-scroll functionality.

## Features

- **Grid Layout**: Configurable columns (1-6) with responsive design
- **Auto-Scroll Carousel**: Smooth fade transitions every 5 seconds
- **Flexible Display**:
  - Position: After Hero section or Before Footer
  - Scope: Homepage only or All pages
- **Rich Content**: Background images, optional logos, titles, descriptions, and links
- **Drag & Drop**: Easy reordering in admin panel
- **Multilingual**: English and Chinese (Simplified) included

## Installation

```bash
composer require quasimo/flarum-ext-carousel-grids
```

## Usage

1. Enable the extension in admin panel
2. Navigate to **Carousel Grids** settings
3. Click **Add Item** to create grid items
4. Configure display settings (columns, position, scope)
5. Click **Save Settings**
